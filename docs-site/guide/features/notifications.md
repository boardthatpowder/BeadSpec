# Notifications

BeadSpec shows system notifications for key events on your issues. Notification preferences are in **Settings → Notifications**.

## Notification types

| Type | Trigger | Default |
|---|---|---|
| Task assigned | A task is assigned to you | On |
| Unblocked | A task blocking yours is resolved | On |
| Comments | Someone comments on your tasks | On |

## Global mute

Toggle **Global mute** in **Settings → Notifications** to suppress all notifications without changing individual settings. This is useful when you need uninterrupted focus — flip it back off to resume notifications.

When global mute is on, all other notification toggles are greyed out.

## Changing preferences

All notification preferences are per-device and stored locally. They are not synced via Dolt.
